# JavaScript developer task

##  Server Side folder

1. in terminal run  `npm i` or `npm install`  to install node_modules

2. run `npm install cors` to install CORS library to allow using api in clint side in the development mode.

3. run `node app.js` to run the server in the development mode. 
Or if you have installed nodemon globaly in your device then run `nodemon`



##  Client Side folder

1. in terminal run  `npm i` or `npm install`  to install node_modules

2. then run `npm start` to run the app in the development mode.
Open http://localhost:3000 to view it in your browser.
